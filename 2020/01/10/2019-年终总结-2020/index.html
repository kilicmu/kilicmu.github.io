<2019>年终总结

January 10, 2020 2273

生活

关键词：平淡

总的来说，这一年就如关键词描述的那样，平淡到炸，每天水水课，考试水水题，大多数时间回到寝室对着电脑，看一些杂七杂八的东西。

很无奈，本来 19 年上半年每天坚持跑步 4-10km，到了下半年也没能坚持到最后，可能是真的没有那份减肥的决心吧。

心态上倒是发生了很大的变化，也许是时间或成长让我更好的接纳了自己，接纳了自己的平庸与各种毛病，所以心态上就变的更为平缓，不会去刻意奢求或在意什么，同时也大体上克服了原来的自卑与讨好，我觉的这可能就是我这 2019 年最大的收获。

2020 年我想要立下一些 flag，因为要考研，可能很多 flag 没有时间去实现，但是我还是想要记录下来，作为一把催促我改变的鞭子：

  • 多看书

  • 做记录

  • 每天做 review

  • 学好英语

  • 更会说话

其实我是个不怎么喜欢看大头书的人，但是我发现在长时间去阅读各中口水博客的过程中，我阅读长篇文章的能力在不断的削弱。包括但不限于浮躁，抓不住重点，无法形成知识体系等。

所以我今年最大的愿望就是可以读三本以上 500 页以上的书籍，处理掉花了很多钱买了很久的数（_ps：说的就是你，《深入理解计算机系统》_）。

英语就不用说了，这方面的弱势简直把我坑惨了，很多文档也无法阅读，简直就是一块心病，今年哪怕其他几项 flag 都失败了也必须要拿下英语。

最后照例祝福一下自己（ps：因为没有其他人会看了_）：新的一年万事如意吧，虽然可能没用（_ps：买票就我一个被人落下，现在还不知道该怎么回家，555….)

技术

关键词：沉痼自若

总结一下这一年，其实我接触了很多的东西，一开始是因为 python 的火热，让我把更多的经历放在了 python 的学习上，在接触了 Django 与一些数据处理技术后，我发现 python 真的有一些局限性。

运行速度暂且不表，很多隐形的错误由于没有编译过程，都是只有在运行过程才能被发现，没有类型限制（3.6 加入了类型标注）真的是写时一时爽，维护火葬场，没有良好的注释情况下，很多函数时间一久就自己都不记得如何传参。动态参数更是让我毛骨悚然，所以我就转去看了 C++。

说是看 C++，但是感觉自己更多的是看了很多 C 的东西，比如 linux 编程与网络编程，更多的是去了解了一些 Unix 的机制和 api，对于 C++的了解真的就是浅尝辄止了，因为这时候我遇到了 go。

不管大家怎么诋毁 golang，由于很多规定，比如大括号不能换行，本来就符合我平日习惯，所以写 go 的代码真的是很爽，当然确实有很多缺陷，比如错误处理与逻辑混合在一起，包管理工具没有那么好用，周边生态完全无法和很多老牌语言比较，但是这不妨碍我对它的喜欢，较少的关键字完全没有妨碍它强大的表达能力，相较其他面向对象语言，没有继承让 golang 的代码更加符合合成聚合的思想，goroutine+channel 的体验很棒，吐槽最多的泛型听说也会在 go2.0 加入，还有什么不知足的呢。

语言只是工具但是还是会限制你的表达能力，所以我选择深入研究 js 去了，原因如下：

因为区块链的火热，我就莫名奇妙去了解了区块链接触了 web3.js，更是掉入了前端的大坑，接触了 Vue 和 React，最近在啃 ES6 以后的各种奇淫巧技，不出意外新的一年我会集中更多的精力在前端上。

至于区块链，其实我感觉就像是个明星技术，看似高大上其实确实有很多问题没有解决。

社会层面，大家都想着用公链包装概念抄袭代码，发币骗钱，把币圈搞的乌烟瘴气，逛一圈论坛以为自己进了黑市，真真假假，尔虞我诈，很多帖子更是酷似传销洗脑，大佬疯狂收割韭菜，韭州大陆名不虚传，不过政策下来后感觉受到了一些控制。

技术层面，如以太坊交易的速度我就不敢恭维了，同时因为是年轻的技术，很多问题资料少点可怜，莫名其妙的错误可能会困扰很久。至于发展速度真是快如火箭，今天还是区块链 2.0，明天就是 3.0，我期待区块链会发展更好，但是它的应用范围真的有限，并且现阶段只能算是起步，所以冷静来看，前景有待静观。

下面总结一些缺点，算是解释一下关键词：

很早以前就发现自己代码写的很混乱，整体代码的可阅读性非常差，仅仅是使用第三方框架然后疯狂堆积业务逻辑。同时，依然不擅长读其他人的优秀源码，同时有一种对自己与大佬们天堑差距的焦虑。

所以我自己新一年我会放平心态，花费更多时间在优秀代码与架构的阅读上，先去模仿学习造一些轮子，争取新的一年会有大的飞跃。

总结

虽然文章写的晚了点，但是 2019 算是过去了，想到 2020 仿佛是儿时科幻影片的数字，如今却蓦然到访，也只能感慨时光飞逝，转眼从十几岁的肥宅变成了一个二十几的肥宅真的是只能感叹岁月无情。

虽早就已经厌倦了象牙塔生活，但出于某些或怯懦或不自信的想法，我终于还是决定要考研了，希望我能完成上面立下的 flag 的同时，还能在明年的三月考研上岸吧。

言至于此，我们 2021 见。
